Prednisone equivalency

convert_pred_equiv( drug_df, drug_col = "drug_name_std", dose_col = "daily_dose_mg_imputed", equiv_table = NULL, drug_name_map = NULL )

Multiplies each record's daily dose by the appropriate prednisone-equivalency factor:

DrugFactorNotes
Prednisone1.0Reference drug
Prednisolone1.0Equipotent
Methylprednisolone1.251 mg methylpred = 1.25 mg pred
Triamcinolone1.25
Dexamethasone7.5High potency
Hydrocortisone0.25Low potency
BudesonideNARoute-dependent; flagged for manual review

Pass a custom equivalency table via equiv_table to override or extend the built-in table. The table must have columns drug_name_std and equiv_factor.

Output columns added

ColumnDescription
pred_equiv_mgPrednisone-equivalent daily dose in mg
equiv_factorMultiplication factor applied
pred_equiv_status"ok", "no_equiv" (drug not in table), "na_dose" (dose was NA), "na_factor" (budesonide etc.)

Episode building

build_episodes( drug_df, id_col = "person_id", drug_col = "drug_name_std", start_col = "drug_exposure_start_date", end_col = "drug_exposure_end_date", dose_col = "daily_dose_mg_imputed", gap_days = 30L )

Merges overlapping or adjacent prescription records within a patient–drug group into continuous treatment episodes using the OHDSI standard 30-day gap. Records are sorted by start date; any two records whose gap is ≤ gap_days are merged into a single episode.

Output columns (one row per episode)

ColumnDescription
person_idPatient identifier
drug_name_stdStandardised drug name
episode_idInteger episode index within each patient–drug pair
episode_startStart date of the episode
episode_endEnd date of the episode
n_daysDuration of episode in days
n_recordsNumber of prescriptions merged
median_daily_doseMedian prednisone-equivalent daily dose across records
min_daily_doseMinimum daily dose in episode
max_daily_doseMaximum daily dose in episode
mean_daily_dose v0.1.3Duration-weighted mean daily dose: sum(dose_i × days_i) / sum(days_i)

run_pipeline()

Convenience wrapper that chains all steps (fetch → impute → convert → episodes) in a single call.

run_pipeline( connector_or_df, method = c("baseline", "nlp"), m2_sig_parse = c("auto", "warn", "nlp_first", "none"), drug_concept_ids = NULL, person_ids = NULL, start_date = NULL, end_date = NULL, sig_source = "sig", gap_days = 30L, equiv_table = NULL, return_level = c("episode", "exposure") )
ParameterDefaultDescription
method"baseline"Imputation method. Use "nlp" for SIG-text parsing.
m2_sig_parse v0.1.8"auto"M2 SIG-parse strategy for Baseline method:
"auto" (default) — call parse_sig() internally to populate tablets/freq_per_day
"warn" — emit warning and skip M2
"nlp_first" — run full NLP pass before baseline (NLP columns retained)
"none" — silently skip M2
gap_days30LGap threshold for episode building.
return_level"episode""episode" = one row per episode; "exposure" = one row per drug-exposure record

Step-by-step alternative

# Equivalent to run_pipeline(con, method = "baseline", return_level = "episode")
dose_df  <- calc_daily_dose_baseline(con, m2_sig_parse = "auto")
equiv_df <- convert_pred_equiv(dose_df,
              drug_col = "drug_name_std",
              dose_col = "daily_dose_mg_imputed")
episodes <- build_episodes(
  equiv_df |> dplyr::filter(!is.na(pred_equiv_mg)),
  end_col  = "drug_exposure_end_date",
  dose_col = "pred_equiv_mg",
  gap_days = 30L
)

Evaluation against a gold standard

evaluate_against_gold( computed_df, gold_df, min_overlap_days = 1L, computed_id_col = "person_id", computed_start_col = "episode_start", computed_end_col = "episode_end", computed_dose_col = "median_daily_dose", gold_id_col = "patient_id", gold_start_col = "episode_start", gold_end_col = "episode_end", gold_dose_col = "median_daily_dose", dose_breaks = c(0, 10, 20, 40, Inf), dose_labels = c("Low (<=10mg)", "Medium (10-20mg)", "High (20-40mg)", "Very High (>40mg)") )

Matches computed episodes to gold-standard episodes using a date-overlap join (any overlap ≥ min_overlap_days). For each gold episode the computed episode with the most overlap is selected.

Key parameters

ParameterDefaultDescription
min_overlap_days1LMinimum calendar-day overlap required to match a computed episode to a gold episode.
computed_dose_col"median_daily_dose"Dose column to compare against gold. Can be "min_daily_dose", "max_daily_dose", or "mean_daily_dose".
dose_breaks v0.1.6c(0, 10, 20, 40, Inf)Cut-points for dose-range stratification in $stratified. Default bins are tuned for typical myositis maintenance doses.
dose_labels v0.1.6c("Low (≤10mg)", …)Labels for the dose ranges. Change both parameters together to customise for other diseases.

Return value

A named list with three elements:

  • $summary — One-row summary: n_gold_periods, n_matched_periods, coverage_pct, MAE, MBE, RMSE, median_AE, MAPE, pearson_corr, spearman_corr
  • $comparison — One row per gold episode: gold_dose, computed_dose, overlap_days, absolute_error, bias_error, agreement_category, error_direction
  • $stratified — Metrics broken down by dose range and taper status

Agreement categories

CategoryCondition
Exact (≤5%)|computed − gold| / gold ≤ 5%
Good (≤20%)≤ 20%
Moderate (≤50%)≤ 50%
Poor (>50%)> 50%

Gold standard standardization

Before calling evaluate_against_gold(), ensure the gold standard median_daily_dose column is in prednisone-equivalent mg/day — the same scale as the computed episodes produced by run_pipeline(). Gold standard records are often in native drug units (e.g., methylprednisolone 8 mg rather than 10 mg pred-equiv). Identify the drug for each gold episode by overlapping it with drug_df, then apply convert_pred_equiv(): 

# identify the dominant drug for each gold episode
gold_drug_map <- drug_df |>
  left_join(gold_std |> select(patient_id, episode_start, episode_end),
            by = c("person_id" = "patient_id"), relationship = "many-to-many") |>
  filter(as.Date(drug_exposure_start_date) <= as.Date(episode_end),
         as.Date(drug_exposure_end_date)   >= as.Date(episode_start)) |>
  group_by(patient_id = person_id, episode_start, episode_end) |>
  count(drug_name_std, sort = TRUE) |>
  slice_max(n, n = 1L, with_ties = FALSE) |> ungroup()

gold_std <- gold_std |>
  left_join(gold_drug_map, by = c("patient_id", "episode_start", "episode_end")) |>
  convert_pred_equiv(drug_col = "drug_name_std", dose_col = "median_daily_dose",
                     out_col = "gold_pred_equiv_mg") |>
  mutate(median_daily_dose = coalesce(gold_pred_equiv_mg, median_daily_dose))

Example

gold_std    <- readr::read_csv(file.path(extdata, "synthetic_gold_standard.csv"),
                             show_col_types = FALSE)
eval_result <- evaluate_against_gold(episodes, gold_std)

s <- eval_result$summary
sprintf("Coverage: %.1f%%  MAE: %.2f  MBE: %.2f  RMSE: %.2f",
        s$coverage_pct, s$MAE, s$MBE, s$RMSE)
